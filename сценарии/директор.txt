Просмотреть кассира
1. Пользователь находится в главном меню
2. Пользователь нажал на кнопку список кассиров
3. Пользователь находится в списке кассиров
#Если СУБД не открывается, то выводится окно с сообщением об ошибке


Добавить кассира
1. Пользователь находится в списке кассиров
2. 
-Пользователь нажал на кнопку добавить
-Открывается форма ввода данных
-Пользователь вводит данные кассира
-Пользователь нажимает сохранить
-Программа добавляет кассира в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке кассиров
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если кассир с такими данными в БД уже существует, то выводится окно с сообщением "такой кассир уже существует"


Удалить кассира
1. Пользователь находится в списке кассиров
2. 
-Пользователь выбрал кассира(ов)
-Пользователь нажал на кнопку удалить
-Программа выводит окно с подтверждением
-Пользователь нажал на кнопку подтвердить
-Программа удаляет кассира(ов) из базы данных
3. Пользователь находится в обновленном списке кассиров
#Если пользователь случайно удалил не то что нужно, то можно использовать сочетание клавиш ctrl+z
#Если пользователь нажал на кнопку отменить, то возвращается в список кассиров с выбранными кассиром(ми)


Поиск кассира
1. Пользователь находится в списке кассиров
2. Пользователь ввел в поле для поиска данные кассира
3. Пользователь находится в списке найденных кассиров
#Если программа ничего не нашла, то выводит сообщение "по вашему запросу ничего не найдено"
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле


Редактировать кассира
1. Пользователь находится в списке кассиров
2. 
-Пользователь выбрал кассира
-Пользователь нажал на кнопку редактировать
-Открывается форма ввода данных, уже заполненная данными выбранного кассира
-Пользователь отредактировал нужные данные
-Пользователь нажал на кнопку сохранить
-Программа сохраняет новые данные в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке кассиров
#Если пользователь выбрал несколько кассиров, то кнопка редактировать будет неактивна (тускнеет)
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если кассир с такими данными в БД уже существует, то выводится окно с сообщением "такой кассир уже существует"

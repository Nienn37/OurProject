# - ошибики
@ - альтернативный сценарий

Просмотреть диски
1. Пользователь находится в главном окне
2. Пользователь выбирает раздел список дисков
3. Пользователь находится в окне списка дисков
#Если СУБД не открывается, то выводится окно с сообщением об ошибке

Добавить диск
1. Пользователь находится в окне списка дисков
2. 
-Пользователь нажимает на кнопку добавить
-Открывается форма ввода данных
-Пользователь вводит данные диска
-Пользователь нажимает сохранить
-Программа выводит окно подтверждения
-Пользователь нажимает на кнопку подтвердить
-Программа добавляет диск в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном окне списка дисков
#Если пользователь вводит неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполняет все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если диск с такими данными в БД уже существует, то выводится окно с сообщением "такой диск уже существует"
#Если СУБД не открывается, то выводится окно с сообщением об ошибке
@Если пользователь нажимает кнопку отмена, то окно подтверждения закрывается

Удалить диск
1. Пользователь находится в окне списка дисков
2. 
-Пользователь выбирает диск(и)
-Пользователь нажимает на кнопку удалить
-Программа выводит окно подтверждения
-Пользователь нажимает на кнопку подтвердить
-Программа удаляет диск(и) из базы данных
3. Пользователь находится в обновленном окне списка дисков
#Если пользователь случайно удаляет не то что нужно, то можно использовать сочетание клавиш ctrl+z.
#Если пользователь нажимает на кнопку отменить, то возвращается в окно списка дисков с выбранными дисками
#Если СУБД не открывается, то выводится окно с сообщением об ошибке
@Если пользователь нажимает кнопку отмена, то окно подтверждения закрывается

Поиск диска
1. Пользователь находится в окне списка дисков
2. Пользователь вводит в поле для поиска данные диска
3. Пользователь находится в окне найденных дисков
#Если программа ничего не находит, то выводится сообщение "по вашему запросу ничего не найдено"
#Если пользователь вводит неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если СУБД не открывается, то выводится окно с сообщением об ошибке

Редактировать диск
1. Пользователь находится в окне списка дисков
2. 
-Пользователь выбирает диск
-Пользователь нажимает на кнопку редактировать
-Открывается форма ввода данных, уже заполненная данными выбранного диска
-Пользователь редактирует нужные данные
-Пользователь нажимает на кнопку сохранить
-Программа выводит окно подтверждения
-Пользователь нажимает на кнопку подтвердить
-Программа сохраняет новые данные в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списка дисков
#Если пользователь выбирает несколько дисков, то кнопка редактировать будет неактивна (тускнеет)
#Если пользователь вводит неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполняет все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если диск с такими данными в БД уже существует, то выводится окно с сообщением "такой диск уже существует"
#Если СУБД не открывается, то выводится окно с сообщением об ошибке
@Если пользователь нажимает кнопку отмена, то окно подтверждения закрывается

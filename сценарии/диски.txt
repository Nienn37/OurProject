Просмотреть диски
1. Пользователь находится в главном меню
2. Пользователь нажал на кнопку список дисков
3. Пользователь находится в списке дисков
#Если СУБД не открывается, то выводится окно с сообщением об ошибке

Добавить диск
1. Пользователь находится в списке дисков
2. 
-Пользователь нажал на кнопку добавить
-Открывается форма ввода данных
-Пользователь вводит данные диска
-Пользователь нажимает сохранить
-Программа добавляет диск в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке дисков
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если диск с такими данными в БД уже существует, то выводится окно с сообщением "такой диск уже существует"


Удалить диск
1. Пользователь находится в списке дисков
2. 
-Пользователь выбрал диск(и)
-Пользователь нажал на кнопку удалить
-Программа выводит окно с подтверждением
-Пользователь нажал на кнопку подтвердить
-Программа удаляет диск(и) из базы данных
3. Пользователь находится в обновленном списке дисков
#Если пользователь случайно удалил не то что нужно, то можно использовать сочетание клавиш ctrl+z.
#Если пользователь нажал на кнопку отменить, то возвращается в список дисков с выбранными дисками

Поиск диска
1. Пользователь находится в списке дисков
2. Пользователь ввел в поле для поиска данные диска
3. Пользователь находится в списке найденных дисков
#Если программа ничего не нашла, то выводит сообщение "по вашему запросу ничего не найдено"
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле

Редактировать диск
1. Пользователь находится в списке дисков
2. 
-Пользователь выбрал диск
-Пользователь нажал на кнопку редактировать
-Открывается форма ввода данных, уже заполненная данными выбранного диска
-Пользователь отредактировал нужные данные
-Пользователь нажал на кнопку сохранить
-Программа сохраняет новые данные в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке дисков
#Если пользователь выбрал несколько дисков, то кнопка редактировать будет неактивна (тускнеет)
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если диск с такими данными в БД уже существует, то выводится окно с сообщением "такой диск уже существует"

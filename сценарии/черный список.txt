Просмотреть черный список
1. Пользователь находится в главном меню
2. Пользователь нажал на кнопку черный список
3. Пользователь находится в черном списке клиентов
#Если СУБД не открывается, то выводится окно с сообщением об ошибке

Добавить в черный список
1. Пользователь находится в черном списке клиентов
2. 
-Пользователь выбрал клиента(ов)
-Пользователь нажал на кнопку добавить в черный список 
-Программа добавляет клиента(ов) в черный список 
3. Пользователь находится в обновленном черном списке клиентов
#Если клиент уже находится в черном списке, то выводится окно с сообщением в котором выводит клиентов, которые уже находятся в чёрном списке


Убрать из черного списка
1. Пользователь находится в черном списке клиентов
2. 
-Пользователь выбрал клиента(ов)
-Пользователь нажал на кнопку убрать из черного списка
-Программа убирает клиента(ов) из черного списка
3. Находимся в обновленном чёрном списке
#Если пользователь случайно убрал не то что нужно, то можно использовать сочетание клавиш ctrl+z


Поиск в черном списке
1. Пользователь находится в черном списке клиентов
2. Пользователь ввел в поле для поиска данные клиента
3. Пользователь находится в списке найденных клиентов 
#Если программа ничего не нашла, то выводит сообщение "по вашему запросу ничего не найдено"
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле

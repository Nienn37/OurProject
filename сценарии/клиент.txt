Просмотреть клиента
1. Пользователь находится в главном меню
2. Пользователь нажал на кнопку список клиентов
3. Пользователь находится в списке клиентов
#Если СУБД не открывается, то выводится окно с сообщением об ошибке


Добавить клиента
1. Пользователь находится в списке клиентов
2. 
-Пользователь нажал на кнопку добавить
-Открывается форма ввода данных
-Пользователь вводит данные клиента
-Пользователь нажимает сохранить
-Программа добавляет клиента в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке клиентов
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если клиент с такими данными в БД уже существует, то выводится окно с сообщением "такой клиент уже существует"
#Если пользователь ввел неправильный формат номера телефона, то выводится окно с сообщением "Некорректный номер телефона"


Удалить клиента
1. Пользователь находится в списке клиентов
2. 
-Пользователь выбрал клиента(ов)
-Пользователь нажал на кнопку удалить
-Программа выводит окно с подтверждением
-Пользователь нажал на кнопку подтвердить
-Программа удаляет клиента(ов) из базы данных
3. Пользователь находится в обновленном списке клиентов
#Если пользователь случайно удалил не то что нужно, то можно использовать сочетание клавиш ctrl+z.
#Если пользователь нажал на кнопку отменить, то возвращается в список клиентов с выбранным(и) клиентом(ами)


Поиск клиента
1. Пользователь находится в списке клиентов
2. Пользователь ввел в поле для поиска данные клиента
3. Пользователь находится в списке найденных клиентов
#Если программа ничего не нашла, то выводит сообщение "по вашему запросу ничего не найдено"
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле


Редактировать клиента
1. Пользователь находится в списке клиентов
2. 
-Пользователь выбрал клиента
-Пользователь нажал на кнопку редактировать
-Открывается форма ввода данных, уже заполненная данными выбранного клиента
-Пользователь отредактировал нужные данные
-Пользователь нажал на кнопку сохранить
-Программа сохраняет новые данные в базу данных
-Форма ввода данных закрывается
3. Пользователь находится в обновленном списке клиентов
#Если пользователь выбрал несколько клиентов, то кнопка редактировать будет неактивна (тускнеет)
#Если пользователь ввёл неправильные символы, то выводится сообщение с разрешёнными символами в поле
#Если пользователь не заполнил все обязательные поля, то выводится окно с сообщением "не все обязательные поля заполнены"
#Если клиент с такими данными в БД уже существует, то выводится окно с сообщением "такой клиент уже существует"
